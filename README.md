### ✨ عن المشروع

تم تطوير هذا الموقع ليكون **دليل فعاليات الجامعة الافتراضية السورية (SVU Events Guide)**، وهو تطوير كامل وإعادة هيكلة جذرية لمشروع **City Events Guide** السابق.

المشروع السابق كان موقعاً أمامياً ثابتاً مستلهماً من الهوية البصرية لموقع وزارة الخارجية والمغتربين السورية، ومبنياً باستخدام **HTML, CSS, JavaScript, jQuery** فقط.

أما هذا المشروع، فقد تمت **إعادة هيكلته بالكامل** ليتحول إلى **نظام إدارة محتوى متكامل (CMS)** مطابق للمعايير العالمية، مع **لوحة تحكم ديناميكية (Admin Dashboard)** تتيح إدارة الفعاليات (إضافة، تعديل، حذف)، ودعم **ثنائي اللغة (العربية والإنجليزية)**، ونظام **جلسات ومصادقة**، واتصال **قاعدة بيانات PDO آمن**.

### 🛠️ التقنيات المستخدمة

- **Frontend:** HTML5, CSS3, JavaScript, jQuery
- **Backend:** PHP 8, PDO, MySQL
- **Tools:** phpMyAdmin, Apache, XAMPP, Git, GitHub

### 🎨 الهوية البصرية والإلهام

يستند هذا المشروع في أساسه البصري إلى مشروع City Events Guide السابق، والذي كان تصميمه مستلهماً من الهوية البصرية لموقع وزارة الخارجية والمغتربين السورية. مع تعديلات جوهرية طالت لوحة الألوان وهيكل الواجهات لتتناسب مع طبيعة المنصة الجامعية. لوحة الألوان الحالية مستمدة من الرموز الوطنية وتم انتقاؤها لتحقيق التوازن البصري والوضوح، مع الحفاظ على طابع رسمي وعصري في آنٍ واحد.

### 🔤 الخط المستخدم

الخط المستخدم هو **"itf Qomra Arabic"**، وهو خط مدفوع ومرخص تصل قيمته التقديرية إلى حوالي **1,500 دولار أمريكي**. تم تضمينه في المشروع بصيغة Base64 مشفرة للحفاظ على حقوق الملكية الفكرية وضمان الامتثال لشروط الترخيص.

### 🏗️ البناء من الصفر

من منظور تقني، لم يكن الإلهام من موقع الوزارة تقليداً، بل انعكاساً لطموحي الشخصي لأكون مطوراً عالي المستوى يلتزم بأعلى المعايير المهنية. لهذا السبب، تم بناء المشروع **بالكامل من الصفر دون الاعتماد على أي أطر عمل جاهزة مثل Bootstrap**، لإظهار التحكم الكامل بالتفاصيل الدقيقة وتقديم تجربة مستخدم فريدة ومخصصة.

### 📜 ملاحظة هامة

يُرجى من المراجعين والمقيمين أخذ هذه النقاط بعين الاعتبار، والامتناع عن استخدام أو إعادة توزيع أي من الأصول المحمية خارج نطاق هذا المشروع.

## 📌 English | EN

### ✨ About the Project

This website was developed as the **Syrian Virtual University Events Guide (SVU Events Guide)**, a complete re-engineering and fundamental restructuring of the previous **City Events Guide** project.

The previous project was a static front-end website inspired by the visual identity of the Syrian Ministry of Foreign Affairs and Expatriates website, built using only **HTML, CSS, JavaScript, and jQuery**.

This project, however, has been **entirely restructured** into a **fully-fledged Content Management System (CMS)** adhering to international standards, featuring a **dynamic Admin Dashboard** for event management (Create, Edit, Delete), **bilingual support (Arabic/English)**, **session management and authentication**, and **secure PDO database connectivity**.

### 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript, jQuery
- **Backend:** PHP 8, PDO, MySQL
- **Tools:** phpMyAdmin, Apache, XAMPP, Git, GitHub

### 🎨 Visual Identity & Inspiration

This project builds upon the visual foundation of the previous City Events Guide project, whose design was inspired by the visual identity of the Syrian Ministry of Foreign Affairs and Expatriates website. Substantial modifications have been made to the color palette and interface structure to suit the nature of this university platform. The current color palette is derived from national symbols and was carefully selected to achieve visual balance and clarity, while maintaining a formal yet modern aesthetic.

### 🔤 Font

The font used is **"itf Qomra Arabic"**, a paid and officially licensed typeface with an estimated value of approximately **$1,500 USD**. It has been embedded in the project in encrypted Base64 format to preserve intellectual property rights and ensure compliance with licensing terms.

### 🏗️ Built From Scratch

From a technical perspective, the inspiration drawn from the Ministry's website was not mere imitation, but rather a reflection of a personal ambition to become a high-level developer who adheres to the highest professional standards. For this reason, the project was built **entirely from scratch without relying on any pre-made frameworks such as Bootstrap**, in order to demonstrate full control over fine-grained details and deliver a uniquely customized user experience.

Reviewers and evaluators are kindly requested to take these points into consideration and to refrain from using or redistributing any protected assets outside the scope of this project.

## 📦 التثبيت والتشغيل | Installation

1.  استورد ملف `config/svu_events.sql` إلى MySQL.
2.  عدل بيانات الاتصال في `config/db.php`.
3.  ارفع جميع الملفات إلى مجلد `htdocs` (XAMPP) أو `public_html` (cPanel).
4.  بيانات المشرف الافتراضية: `admin` / `admin`.
